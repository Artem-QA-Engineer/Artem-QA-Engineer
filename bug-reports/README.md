# Баг-репорт

**Заголовок**: Фото товаров не соответствует реальным продуктам на странице после авторизации под логином "problem_user"

**Приоритет**: High

**Серьезность/ критичность**: Critical

**Шаги воспроизведения**:
  1. Открыть страницу сайта https://www.saucedemo.com/
  2. В блоке Username ввести problem_user
  3. В блоке Password ввести secret_sauce
  4. Нажать на кнопку Login

**Ожидаемый результат**: пользователь переходит на страницу с товарами и видит актуальные фото предоставляемых товаров

**Фактический результат**: пользователь переходит на страницу с товарами и видит фото собаки

**Окружение:** Win 11, Google Chrome Версия 145.0.7632.160 (64 бит)
