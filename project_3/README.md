# Тестирование API httpbin.org через Postman

Тестирование публичного API [httpbin.org](https://httpbin.org/). Сервис, который используется для отладки HTTP-запросов.

## Что сделано

- Создана коллекция из 3 запросов (GET, POST, DELETE)
- Созданы сниппеты на JavaScript для проверки статус-кодов
- Коллекция экспортирована в JSON, готова к импорту без дополнительных настроек

## Запросы

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | /status/200 | Проверка статус-кода 200 |
| POST | /post | Отправка JSON-тела и проверка ответа |
| DELETE | /delay/5 | Проверка отложенного ответа (5 секунд) |

## Сниппеты

К каждому запросу добавлен сниппет проверки статус-кода.
