## Результат:

**Основная проблема:** в ходе тестирования было выявлено, что отсутствует валидация длины номера телефона. Пользователь может, например, не дописать одну цифру в номере телефона, но заявка при этом будет успешно оформлена. За счет этого менеджер не сможет дозвониться лиду, что **приведет к потере прибыли для бизнеса.**
