# Тест-кейсы заполнения заявки на обратный звонок сайта https://fitness-cccp.ru

Общее предусловие для всех тест-кейсов: 
1. Пользователь находится на сайте https://fitness-cccp.ru
2. Пользователь нажал на сайте вверху справа на кнопку "4 подарочные тренировки"
3. Перед пользователем открылась заявка на обратный звонок

| ID тест-кейса | Заголовок | Шаги | Ожидаемый результат | Статус проверки (Pass/ Fail) |
|---|---|---|---|---|
| 01 | Отправление заявки при всех валидных данных | 1. Ввести валидные данные в поле "имя" (например, Иван)<br> 2. Ввести валидные данные в поле "номер телефона" (например, +7-911-111-11-11)<br> 3. Ввести валидные данные в поле "эл. почта" (например, example@mail.ru)<br> 4. Выбрать любой клуб в поле "выберите клуб"<br> 5. Ввести в поле "ваше сообщение" текст до 100 символов (например, "позвоните в пятницу")<br> 6. Отметить чек-бокс согласия с обработкой персональных данных<br> 7. Нажать кнопку "отправить заявку"<br>  | 1. Заявка успешно отправилась<br> 2. Появилось сообщение "заявка успешно отправлена"<br> 3. После успешной отправки форма заявки становится пустой | Pass |
| 02 | Отправление заявки с пустыми полями | 1. Все поля оставить пустыми<br> 2. Нажать кнопку "отправить заявку"<br> | 1. Заявка не отправилась<br> 2. Пустые обязательные поля подсвечиваются красным <br> 3. У незаполненных обязательных полей появляются надписи "обязательное поле" | Pass |
