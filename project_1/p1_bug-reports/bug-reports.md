# Баг-репорты по заполнению заявки на обратный звонок сайта https://fitness-cccp.ru

## Баг-репорт №1 
**Заголовок**: Поле "номер телефона" успешно принимает номер из менее 11 цифр 

**Приоритет**: High

**Серьезность/ критичность**: Major

**Шаги воспроизведения**:
  1. Открыть страницу сайта https://fitness-cccp.ru
  2. Нажать на сайте вверху справа на кнопку "4 подарочные тренировки"
  3. В заявке на обратный звонок ввести валидные данные в поле "имя" (например, Иван)
  4. Ввести номер телефона из 10 цифр в поле "номер телефона" (например, +7-911-111-11-1)
  5. Выбрать любой клуб в поле "выберите клуб"
  6. Отметить чек-бокс согласия с обработкой персональных данных
  7. Нажать кнопку "отправить заявку"

**Ожидаемый результат**: 
  1. Заявка не отправилась
  2. Поле "номер телефона" подсвечивается красным
  3. У поля "номер телефона" появилась надпись "номер телефона должен содержать 11 цифр"

**Фактический результат**: Заявка отправилась

**Окружение:** Win 11, Google Chrome Версия 145.0.7632.160 (64 бит)
