# Тестирование мобильного трафика через Charles Proxy

Перехват и модификация трафика на реальных сайтах через Charles Proxy на эмуляторе Android Studio

## Что сделано

- Настроен Charles Proxy (сертификат на эмуляторе Android)
- Освоены ключевые функции сниффера
- Каждая функция отработана на отдельном сайте
- Записаны скринкасты

## Реализованные задачи

| Функция | Сайт | Задача |
|---|---|---|
| Breakpoints | Ситилинк | Изменение названия товара |
| Map Local | demoshopping.ru | Подмена картинки товара на локальную |
| Rewrite | happywear.ru | Подмена цены товара на 1 рубль |
