# Привет!
Я начинающий специалист по тестированию. В этом репозитории собраны примеры моей тестовой документации.

## Навыки и инструменты
- Тест-дизайн: классы эквивалентности, граничные значения
- Документация: чек-листы, тест-кейсы, баг-репорты, тест-план
- Инструменты: GitLab, GitHub, Postman, Charles Proxy, DevTools, Android Studio, VS Code, InteliJ IDEA, PyCharm, pgAdmin, Jira, Яндекс Трекер
- Языки программирования: основы SQL, Java, Python, C, Bash
- Дополнительно: понимание клиент-серверной архитектуры, REST API

## Проекты:
- **[Ручное тестирование веб-формы: чек-лист, тест-кейсы и баг-репорты](./project_1/)** — функциональное тестирование формы заявки на реальном сайте [fitness-cccp.ru](https://fitness-cccp.ru/). Чек-лист из 45 проверок, 9 тест-кейсов, 4 баг-репорта. Использован Chrome DevTools.
- **[Тестирование мобильного трафика через Charles Proxy](./project_2/)** — перехват и модификация трафика на реальных и учебных сайтах через Charles Proxy на эмуляторе Android Studio 
- **[Тестирование API с помощью Postman](./project_3/)** — коллекция запросов, автотесты (httpbin.org)
